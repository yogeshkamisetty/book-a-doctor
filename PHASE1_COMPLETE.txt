🎓 HOW TO GET STARTED
═══════════════════════════════════════════════════════════════════════════════

1. START BACKEND
   $ cd backend
   $ npm install (if not done)
   $ npm run dev
   → Backend running on http://localhost:5000

2. START FRONTEND
   $ cd frontend
   $ npm install (if not done)
   $ npm run dev
   → Frontend running on http://localhost:3000
